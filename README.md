# Modern Portfolio Template

A clean, modern, and responsive portfolio website template for developers, designers, and creatives. Built with performance, accessibility, and simplicity in mind, making it easy to showcase your projects, skills, experience, and contact information.

## Features

* Clean and minimalist design
* Fully responsive layout
* Fast loading and optimized
* Project showcase section
* Skills & technologies section
* Awards & recognition section
* Contact form with SMTP support
* Easy to customize
* Environment variable support

---

## Built With

* HTML5
* CSS3 / SCSS
* JavaScript
* PHP
* PHPMailer

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ganeshbistakaji/simple-folio.git
cd simple-folio
```

### 2. Install PHP dependencies

```bash
composer install
```

### 3. Create your environment file

Create a `.env` file in the project root.

Example:

```env
# SMTP Configuration
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=sender_email
SMTP_PASS=app-password
CONTACT_EMAIL=receiver_email
```
